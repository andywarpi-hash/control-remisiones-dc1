Integración con Google Sheets vía Apps Script Web App

🔗 URL del Web App (Google Apps Script)

📋 Estructura esperada del Google Sheets

El Apps Script debe retornar un JSON con un array de objetos. Columnas esperadas:

COLUMNAS REQUERIDAS

• Col A — N° Remisión • Col B — Coordinador (quién crea la OC) • Col C — Contratista • Col D — Tienda / PDV • Col E — Días creación OC (coord.) • Col F — Fecha de creación OC • Col G — Días radicación (contratista) • Col H — Fecha radicación • Col I — Estado • Col J — Observaciones

APPS SCRIPT BÁSICO

function doGet() {
  const ss = SpreadsheetApp
    .getActiveSpreadsheet();
  const sh = ss.getActiveSheet();
  const rows = sh.getDataRange()
    .getValues();
  const [header, ...data] = rows;
  const result = data.map(r => {
    let obj = {};
    header.forEach((h,i) => {
      obj[h] = r[i];
    });
    return obj;
  });
  return ContentService
    .createTextOutput(
      JSON.stringify(result))
    .setMimeType(
      ContentService.MimeType.JSON);
}
